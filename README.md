🌤️ Sky Cast Weather App

Sky Cast is a modern and responsive weather application that provides real-time weather information and a 7-day forecast for any city worldwide. Built using HTML, CSS, and JavaScript, it integrates the OpenWeather API to deliver accurate weather data with an interactive and visually appealing user interface.

✨ Features
🌍 Search weather by city name
📍 Get weather using your current location
🌡️ Real-time temperature, humidity, and wind speed
📅 7-Day weather forecast
🎨 Dynamic background based on weather conditions
☁️ Weather animations using Lottie
🌙 Light/Dark theme toggle
📱 Fully responsive design
⚡ Fast and user-friendly interface
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
OpenWeather API
Lottie Animations
